GetValue<T>(IDataReader, String)

Gets the value of the specified column.

Declaration
public static T GetValue<T>(this IDataReader rdr, string columnName)
Parameters
Type Name Description
System.Data.IDataReader rdr

The IDataReader.
System.String columnName

Name of the column.
Returns
Type Description
T

The value of the column as the type specified by the type parameter T.
Type Parameters
Name Description
T

Type for the value.
Remarks

If the value of the column is DbNull, then the default value for the type is returned.

Examples
SqlDataReader rdr = cmd.ExecuteReader();

String customerName = null;
if (rdr.Read())
{
    customerName = rdr.GetValue<String>("customer_name");
}
Exceptions
Type Condition
System.InvalidCastException

Throws an InvalidCastException if the value of the column cannot be cast to type T.
